## Installation
Download the 64-bit android apk from [Gamejolt](https://gamejolt.com/games/sonictripletrouble16bit/322794) and put it in `ports/sonictt16b` as `game.apk`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Pause / Accept|
|D-PAD|Move|
|LEFT ANALOG|Move|
|A|Jump / Fly|
|B|Switch characters|
